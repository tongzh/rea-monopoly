A) HTML Structure - top

This theme is a Responsive layout with two columns generally. All of the information within the main content area is nested within a div with an id of "content". The general template structure is the same throughout the template. Here is the general structure.

<!DOCTYPE html>
<html>
<head>

</head>
<body>
<!-- Start Wrapper-->
<div id="wrapper">
<!-- start breadcrumb -->
<div id="breadcrumb-wrapper">

</div><!-- end breadcrumb -->
<!-- start header-wrapper -->
<div id="header-wrapper">

</div><!-- End header-wrapper -->
<!-- start Navigation -->
<div id="navigation">

</div><!-- start Navigation -->
<div id="content">

</div>
<!-- start Footer -->
<div id="footer-wrapper">

</div><!-- end Footer -->
</div><!-- End Wrapper-->
</body>
</html>

If you would like to edit the color, font, or style of any elements in one of these columns, go to folder proper_theme/css and open files style.css.

C) JavaScript - top

go to folder proper_theme/js and open files proper.js, you can see all plugin config

init_tabs();

/*backgroung Config*/
